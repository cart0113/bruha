# About

I am a software engineer who writes scientific computing software by day and
builds open source tools by night. This blog is where I write about my personal
projects, mostly focused on developer tooling and data analysis.

> [!note] This site is powered by [bruha](https://github.com/cart0113/bruha), a
> set of docsify extensions I built for my documentation and blog sites.

## Projects

- [context-db](https://github.com/cart0113/context-db) — hierarchical Markdown
  knowledge base for LLM agents
- [git-sync](https://github.com/cart0113/git-sync) — compose independent git
  repos into a pseudo-monorepo
- [bruha](https://github.com/cart0113/bruha) — docsify extensions with YAML
  config, themes, and auto-generated sidebars
- [NBA Comeback Dashboard](https://nba-comeback-dashboard.github.io/) — analyze
  NBA comeback probability data from 1996 to now
